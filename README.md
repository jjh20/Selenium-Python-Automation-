# 🚀 Selenium Python Automation Portfolio
Framework profesional de automatización de pruebas E2E.

### 🧪 Proyectos Incluidos:
1. **SauceDemo E2E:** Flujo completo de compra (Login -> Carrito -> Checkout -> Finish).

### 🛠️ Retos Técnicos Superados:
- Manejo de notificaciones nativas de Chrome.
- Clics forzados con JavaScript para evitar interrupciones de seguridad.
- Arquitectura basada en Pytest Fixtures (`conftest.py`).
- Configuración de DNS para descarga de Drivers.

### ⚙️ Cómo ejecutar:
1. `pip install -r requirements.txt`
2. `python -m pytest -v -s`
